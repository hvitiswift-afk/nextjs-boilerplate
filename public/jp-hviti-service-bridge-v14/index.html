Version-control modules

PROFILES

DIFF

Compare two revisions field by field and expose changed values.

ROLLBACK

Restore prior revision

Load an earlier version into the editor and continue from that point.

CHECKSUM

Local integrity marker

Generate a browser-side checksum-like fingerprint for change detection, not cryptographic proof.

AUDIT

Portable history

Export the packet and all versions as one JSON audit artifact.
